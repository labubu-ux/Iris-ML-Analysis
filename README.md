# 📊 Machine Learning Project (Iris)

## 📌 Overview
This project performs data analysis and machine learning to predict outcomes using real-world data.

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🔍 Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Selection
4. Model Building:
   - Logistic Regression
   - Random Forest
5. Model Evaluation:
   - Accuracy Score
   - Confusion Matrix
   
## 🎯 Objective
To classify iris flower species using machine learning models and compare their performance.

## 📊 Models Used
- Logistic Regression
- Random Forest

## 📈 Results
- Random Forest achieved highest accuracy

## 📂 Files Included
- google colab (.ipynb)
- Dataset (.csv)

## 🚀 How to Run
1. Open in Google Colab
2. Run all cells
